Flood fill is an algorithm mainly used to determine a bounded area connected to a given node in a multi-dimensional array. It is a close resemblance to the bucket tool in paint programs.

Fill: It will fill whole grid with color from first color input box.

FloodFill: To start flood-fill.

Reset: It will restore.

InputBox: Enter row number for flood-fill.

InputBox: Enter column number for flood-fill.

ColorInput1: Choose color we want to fill in grid.

ColorInput2: Choose color for flood-fill.

Label: It will show the row number and column where we clicked.
